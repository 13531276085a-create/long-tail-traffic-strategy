# 长尾流量内容策略 Skill

给内容创作者用的 Claude Code skill。输入一个主题，输出长尾关键词矩阵 + 爆款文案 + 合规检查。

## 能做什么

### 趋势调研 · 「现在发什么容易火」
搜小红书/B站/知乎当前热门话题，输出选题方向和本周发布优先级。

> 你：「看看水晶赛道最近什么火」
> 它：热门品类排行 + 爆款内容特征 + 3 个可执行选题 + 发布建议

### 关键词挖掘 · 「挖到大 V 没覆盖的蓝海词」
以核心主题为原点，沿搜索意图、用户旅程、内容角度三个维度展开长尾关键词矩阵，标注搜索量、竞争度和机会值。

> 你：「帮我做紫微斗数的长尾词矩阵」
> 它：20 个精准长尾词 + 每个词的需求缺口分析 + 信息增量切入点 + 优先级排序

### 文案生成 · 「学术风原文直接变爆款贴子」
六项语言风格自动转换：去学术化 → 短句分行 → 金句收尾 → 第二人称共鸣 → 口语化比喻 → 互动钩子。九大标题公式可选，三大内容框架适配不同场景。

> 你：贴一段命理学术原文
> 它：直接出小红书可发文案，标题 + 正文 + 标签，复制就能发

### 合规检查 · 「发之前扫一遍违禁词」
覆盖 60+ 个敏感词，三级风险标注（高危/中危/低危）。独有「标签叠加检测」：单个词安全、组合后高危的情况自动识别。支持小红书/抖音/知乎/B站/公众号/独立站六平台差异化规则。

> 你：「帮我检查这篇有没有违禁词」
> 它：逐项标记 + 替换建议 + 平台适配提醒

### 内容策略 · 「一整月的选题日历」
对于想系统铺一个赛道的创作者，输出完整内容策略：选题日历、话题集群、系列串联方案、差异化切入点分析。

> 你：「帮我规划紫微斗数一个月的内容」
> 它：周更节奏 + 系列选题 + Pillar-Cluster 布局 + 流量闭环设计

### 适配场景

| 领域 | 示例 |
|------|------|
| 命理/玄学 | 紫微斗数、八字、塔罗、星盘 |
| 水晶/疗愈 | 水晶搭配、能量石、情绪消费 |
| 生活方式 | 咖啡、健身、理财、职场 |
| SEO/博客 | 独立站关键词布局、长尾内容策略 |

```

```
