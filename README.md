# 长尾流量内容策略 Skill

给内容创作者用的 Claude Code skill。输入一个主题，输出长尾关键词矩阵 + 爆款文案 + 合规检查。

## 能做什么

- **趋势调研**：搜一下赛道最近什么话题在涨
- **关键词挖掘**：生成精准长尾词矩阵，找需求缺口
- **文案生成**：学术风原文 → 小红书爆款博主口语，带标题公式
- **合规检查**：敏感词扫描 + 标签叠加检测 + 六平台规则

## 安装

```bash
npx skills add 13531276085a-create/long-tail-traffic-strategy@long-tail-traffic-strategy -g -y
```

## 使用

不需要记指令，自然说话就行：

```
看看水晶赛道最近什么火
帮我做紫微斗数的长尾词矩阵
帮我写一篇廉贞化禄的帖子
帮我检查这段文案有没有违禁词
```

## 更新

```bash
npx skills update
```
