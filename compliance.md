# 内容合规与敏感词检测

适用于命理/玄学/健康/金融等敏感赛道。核心原则：**传统文化讨论合法，封建迷信预测违规。区别在于你怎么说，而不是你说什么。**

---

## 使用说明

每次输出内容前，按以下流程自检：

1. **扫描高危词** → 命中任何一个，必须替换，不能保留
2. **扫描中危词** → 评估上下文，能替换则替换，不能则加文化类前置语软化
3. **扫描低危词** → 注意频率，同一篇内容不超过 3 个低危词同时出现
4. **句式中危检测** → 禁止「保证/一定/100%」+ 任何命理结论的组合

## 标签叠加检测（新增 · 核心）

**单个词安全，组合后高危。** 这是最容易漏检的风险。以下组合在同一个 100 字段落内出现，整体视为高危内容：

| 类别 A | 类别 B | 叠加风险 | 示例 |
|--------|--------|---------|------|
| 命理/星曜术语 | 求财/搞钱/赚钱 | 命理+金融预测 | 「廉贞化禄的求财方式」→ 触发玄学诱导消费 |
| 命理/星曜术语 | 投资/股票/基金 | 命理+投资建议 | 「你这个配置适合炒股」→ 非法投资咨询 |
| 命理/星曜术语 | 疾病/症状/治疗 | 命理+医疗诊断 | 「迁移宫有煞星容易得XX病」→ 非法医疗建议 |
| 命理/星曜术语 | 分手/离婚/出轨 | 命理+感情预测 | 「你们明年会离婚」→ 封建迷信 |
| 转运/改运类词 | 任何财富词汇 | 迷信+诈骗 | 「改运后收入翻倍」→ 违规广告 |
| 任何命理词 | 绝对化承诺词 | 虚假宣传 | 「看命盘100%准」→ 必删 |

**处理规则**：命中任何一行 → 保持核心信息但把框架从「A类话术」换成「B类安全话术」。例：「求财模式」→「行事风格」，「搞钱人格」→「做事特点」。

## 三级：低危词（控制频率 ≤ 3次/篇）

| 词汇 | 风险说明 | 替代思路 |
|------|---------|---------|
| 紫微斗数 | 小红书频繁使用触发玄学标签；抖音标题里出现直接进审核队列 | 交替使用「紫微」「这门传统学问」「古代智慧」。抖音标题**不要用**，正文控制在 1 次以内 |
| 命 / 命盘 / 命运 | 宿命感词汇，降低推荐权重 | 交替使用「星盘」「盘面图」「出生信息」 |
| 宫位 / 星曜 | 专业术语可保留，但首次出现需加文化注释 | 「紫微斗数这门传统学问中…」 |
| 四化 / 化禄 / 化忌 | 专业术语相对安全 | 同上 |
| 五行 | 传统文化标签，相对安全 | 不需要替换 |

| 词汇 | 小红书标题 | 小红书正文 | 抖音标题 | 抖音正文 | 知乎/SEO |
|------|-----------|-----------|---------|---------|---------|
| 紫微斗数 | ⚠️ 避免 | 控制在 1-2 次 | 🚫 不要出现 | 控制在 1 次 | ✅ 可正常使用 |

## 句式级安全规则

### 禁止句式（命中整句重写）

| 句式模式 | 示例 | 为什么危险 |
|---------|------|-----------|
| 「你 + 未来预测」 | 「你明年会结婚」 | 直接预测=封建迷信 |
| 「如果 X，就会 Y」 | 「如果你不化解，就会破财」 | 恐吓诱导 |
| 「X 注定 Y」 | 「你注定晚婚」 | 宿命论 |
| 「只要 X，就能 Y」 | 「只要佩戴这个，就能改运」 | 虚假承诺 |
| 「X 说明你一定 Y」 | 「夫妻宫有天相，婚姻一定好」 | 绝对化判断 |

### 安全句式模板（禁止 vs 安全）

| 意图 | 禁止写法 | 安全写法 |
|------|---------|---------|
| 解读命盘 | 「你的命盘显示你会离婚」 | 「这类命盘在感情上通常有这样的特点：……需要多注意沟通方式」 |
| 给建议 | 「你要化解煞星」 | 「当命盘中有这样的配置时，通常建议在XX方面多加留意」 |
| 回应焦虑 | 「不用担心，你不会嫁不出去」 | 「空宫≠不好，只是意味着这个领域有更多可能性」 |
| 展示能力 | 「我算命很准」 | 「命盘解读在传统文化中有几千年的传承，我做的事是帮你看懂自己的命盘结构」 |

### 万能免责句式（每篇至少嵌入一句）

- 「以上内容基于传统文化研究视角，不构成任何预测或承诺」
- 「紫微斗数是传统智慧，但具体的人生方向还是要结合现实情况自己把握」
- 「命盘解读提供的是一个参考框架，具体选择在你自己」
- 「仅供文化探讨，生活中遇到具体问题建议咨询相关专业人士」
- 「以上为出生信息推算的命盘结构分析，不是预测，也不是定论」

## 玄学赛道三大雷区详解（附改写示范）

这三个雷区是玄学内容被封/被举报的最高频原因。每条都给出具体改写逻辑。

### 雷区 1：断言式预测

**触发逻辑**：「你明年3月会发财」→ 你在替用户做确定性承诺 → 一旦不准 = 「骗钱」→ 平台直接判定违规。

| 违规写法 | 安全改写 | 改写逻辑 |
|---------|---------|---------|
| 「你明年春天会升职」 | 「从命盘周期看，明年春天你的官禄宫进入活跃期——不是一定会升，是你在这段时间做的努力被看见的概率更大」 | 断言→趋势分析；责任留在用户手里 |
| 「你的财运在8月」 | 「你命盘里财星最强的月份集中在夏末——适合重点推进项目，但具体能进多少要看前期铺了多少路」 | 不改运、不保证、只讲周期规律 |
| 「你35岁转运」 | 「大多数这种命盘结构的人，35岁前后会经历一个明显的阶段切换——从积累期进入释放期」 | 讲大多数、讲倾向、不讲个体预言 |

**核心操作**：把「你一定会X」改成「从你的命盘结构看，X的窗口期在XX阶段——不是命中注定，是这段时间做什么都会被放大。把握住了是机会，不把握也不会怎样」。

### 雷区 2：因果绑定

**触发逻辑**：「因为你八字缺金，所以你存不住钱」→ 「用迷信解释现实困境」→ 直接踩红线。

| 违规写法 | 安全改写 | 改写逻辑 |
|---------|---------|---------|
| 「你财帛宫空了所以穷」 | 「你的财帛宫是空宫——不是说没钱，是钱财进出的节奏不太稳定。知道这件事之后，不是改命，是在行为层面给自己加一道人工预算阀门」 | 命理描述→行为分析；解方回到现实行动 |
| 「你夫妻宫有煞星所以感情不顺」 | 「夫妻宫里有这颗星的配置，古人认为在亲密关系里容易因为'小事积累'出问题——翻译成现代话就是：定期吵架不如定期复盘」 | 不归因命运、给行为建议 |
| 「你福德宫不好所以你焦虑」 | 「福德宫弱的配置，在传统文化里对应的是'精神燃料消耗快'——你需要比大多数人更刻意地保护自己的情绪能量」 | 把迷信说法翻译成现代心理学语言 |

**核心操作**：把「命里缺X导致你Y」改成「命盘里X的配置有一种倾向——这种倾向对应的现代行为模式是Y。知道了之后不是等改运，是在日常行为里加一道人工检查」。

### 雷区 3：引发恐惧

**触发逻辑**：「你这个大运犯太岁，不化解会出事」→ 恐吓诱导 → 不仅是违规，而且是信任杀手。你今天吓他一次，他明天就不信你了。

| 违规写法 | 安全改写 | 改写逻辑 |
|---------|---------|---------|
| 「你明年犯太岁要化解」 | 「犯太岁的年份，古人叫'值年太岁值班'——太岁的脾气决定了这一年容易在什么领域出摩擦。不是一定倒霉，是这一年你在某些领域要多留一个心眼」 | 恐惧→提醒；化解→多留个心眼 |
| 「迁移宫这颗煞星会让你出行出事」 | 「迁移宫带了这颗星，传统文化里对它的描述是'出门容易遇到计划外的变数'——翻译成现代操作就是：重要行程多做一个备选方案」 | 血光之灾→计划外变数→给具体操作 |
| 「不调理这个宫位健康会出大问题」 | 「这个宫位的配置在古人看来属于'精力输出大于输入'的体质——如果你现在觉得容易累，不是命的问题，是你身体的信号已经发了」 | 把玄学警告转成身体信号的提醒，指向行动而非恐惧 |

**核心操作**：把「不化解会遭殃」改成「古人说这种配置需要在X方面多一份敬畏——现代话说就是多一步检查、多一个备选、多留一个心眼」。恐惧转提醒，花钱化解转行为调整。

## 黑话转换词库（优先使用）

生成内容时遇到敏感词，**优先用本词库的圈内黑话替换**，找不到匹配时才使用前面的正式安全词。

### 风险与避雷

| 敏感词 | 黑话转换 |
|--------|---------|
| 杀猪盘 / 骗局 | 割韭菜、智商税 |
| 套路 | 玩法、操作 |

### 使用规则

1. **同一篇内容同一类词保持一致**：不要一处写「米」一处写「Q」
2. **标题优先用黑话**：标题触犯敏感词时直接换黑话，不改含义
3. **正文首次出现加引号**：比如「搞"米"」，给读者的理解门槛留个缓冲
4. **抖音比小红书更严**：抖音上绿泡泡/公主号/啵啵间本身也可能被识别，慎用

## 平台差异速查

### 小红书

**红线**：禁止算命、占卜、看相、风水、法术仪式；禁止站外导流；禁止封建迷信商业变现。

**允许**：传统文化科普、个人感悟分享（不预测他人）、学术讨论。

**审核机制**：机器关键词扫描 + OCR 封面文字 → 被举报进入人工复审 → 长期玄学内容累积账号标签导致默认限流。

**安全框架**：
- 定位：传统文化研究者 ≠ 命理师
- 标题：紫微斗数怎么看XX ≠ 你的命盘显示你XX
- 正文：带你了解/认识/入门 ≠ 帮你预测/解析/看
- 互动：评论区不回复个人命盘解读，不引导私信
- 封面：不要写「算命」「运势」「改运」

**人设安全公式**：「传统文化科普者」+「自学成才/5年研究」+「不做预测，只做解读」

### 抖音

**审核等级**：全平台最严。封建迷信是重点专项整治类目，封号不需要多次违规——一次严重违规直接永久封禁。

**视觉审核（机器自动识别）**：
- 画面中禁止出现：罗盘、签筒、符咒、八卦镜、香炉、法坛、道袍
- 背景不要有任何宗教/玄学道具（包括书架上的罗盘装饰）
- 手持道具只能在手部以下、不能举到镜头前

**文字审核**：
- 标题：「紫微斗数」「命盘」「运势」「算命」→ 任何玄学术语出现在标题直接进人工审核队列，大概率限流
- 字幕/花字：可与画面配合出现专业术语，但前 3 秒不能出现
- 话题标签：只用「#传统文化」「#国学智慧」，不用「#紫微斗数」「#命理」

**内容审核**：
- 开口前三句必须定调：「今天聊一个传统文化话题」「分享一个古人留下的智慧」
- 视频中途出现玄学术语之前，必须有文化科普的免责铺垫
- 严禁内容：预测他人运势、教人化解/改运、健康预测、投资建议

**直播**：
- 直播算命/连麦看命 → 直接封直播间（不是警告，是直接封）
- 直播间标题和封面不许出现玄学词
- 直播间背景不许出现命盘图

**账号策略**：
- 因为封号风险高，建议**主号 + 备用号同步养**。备用号不发纯玄学内容，发传统文化/国学泛内容积累基础粉丝
- 被封后 24 小时内可申诉，但玄学类账号申诉成功率 < 20%

### 微信公众号

**审核等级**：相对最宽松。公众号属于订阅制，不是推荐制——内容不主动推给非粉丝，所以审核压力比推荐流平台小很多。

**红线**：
- 禁止在菜单栏/自动回复设置「付费算命」「八字预测」「运势占卜」等入口（会被举报）
- 禁止文章内直接导流到微信做"付费算命"（导流到"传统文化交流"可以）
- 禁止发布声称能"化解/改运/治病"的内容

**安全框架**：
- 账号定位：传统文化传播者 / 国学爱好者 ≠ 命理师
- 简介话术：「分享传统文化中的生命智慧」「国学爱好者，研究紫微斗数X年」
- 菜单命名：「命盘入门」「传统文化」≠「算命」「运势预测」
- 付费内容：知识星球/小报童中做深度内容（知识付费 ≠ 算命收费），叫「传统文化研习社」而非「八字算命」

**私域导流安全链路**：
```
文章结尾 → 「想了解更多传统文化知识？关注我，一起交流」
  → 个人微信（定位为「文化同好交流」，不是「算命师傅」）
  → 朋友圈日常内容：80% 传统文化科普 + 20% 个人生活
  → 私聊转化：不做主动推销，等对方先抛出问题
```

**视频号联动**：视频号规则趋近抖音——标题不能用玄学术语，但比抖音稍松。视频号直播间规则跟抖音一致。

### B 站

**审核等级**：比小红书松、比抖音松很多。对知识类/文化类内容天然宽容。

**分区**：
- 必须选「知识」→「人文历史」或「知识」→「社科·法律·心理」
- 选错分区直接限流，且分区选择后不可更改
- 描述/标签可以用「传统文化」「国学」「命盘入门」

**内容安全**：
- 视频开头必须声明：「本视频仅供文化探讨」
- 标题可以使用「紫微斗数」「命盘」——但不能搭配断言式预测（"你的命盘显示你XX"→ 不行）
- B站用户对"中二"调性有天然宽容度——用娱乐化/玩梗的语气讲玄学，比严肃科普更安全

**互动管理**：
- 弹幕区 / 评论区：不回个人命盘解读
- 简介区：放资源链接 + 免责声明 + 不引导"加V算命"
- 可以引导关注（"下期继续聊XX星"），但不能引导付费算命

**直播**：
- B站直播的玄学容忍度高于抖音、低于 YouTube
- 可以聊命盘知识——但不能"连麦看命"，不能被理解为在收费服务
- 直播间标题/封面避免玄学词

### 独立站 / SEO 博客

- Google 不对中文算命内容设限，百度对算命类网站有降权
- 独立站可做最完整的命理内容（不被删帖），内容复利最好的渠道
- 小红书/抖音发「安全版」→ 引导搜索独立站 → 独立站承接深度阅读和咨询转化

## 自检 SOP（已升级）

```
第一步：高危词扫描
  命中一级高危词（算命、改运、保证、斗争、权力等）？
  → 是：立即替换 → 否：进入第二步

第二步：标签叠加检测 ★ 新增
  同一段落内是否存在「命理词 + 财富/疾病/感情/投资词」组合？
  检查清单：求财/搞钱/赚钱 | 投资/股票/基金 | 疾病/治疗 |
             分手/离婚/出轨 | 转运/改运+财富词 | 命理词+绝对化承诺
  → 是：保持核心信息，换安全框架（如「求财模式」→「行事风格」）
  → 否：进入第三步

第三步：中危词密度检查
  中危词密度 > 1个/200字？
  → 是：降低密度或替换 → 否：进入第四步

第四步：句式安全检查
  存在「你 + 未来预测」句式？
  存在「保证/一定/100%」+ 命理结论？
  存在「如果 X 不 Y，就会 Z」恐吓句式？
  → 是：用安全句式模板重写 → 否：进入第五步

第五步：免责声明
  至少嵌入一条免责句式？
  已在人设层面把「命理师」换成「传统文化爱好者」？
  → 是：可以发布 → 否：补上后发布
```

### 模拟举报自测（已升级）

写完内容后，用审核员视角问自己：

1. **标签叠加检查**：这篇内容里，命理词汇旁边有没有财富/感情/健康类词汇？→ 有的话，每个组合都要确认安全
2. **「这篇内容是在科普知识，还是在给人指路？」** → 指路就重写
3. **「截图发给平台客服，能不能活？」** → 不能就重写
